## 1. Description
A prototype of a blog application developed using **ASP.NET MVC 5** and **SQL Server**. The design leverages the **Clean Blog** template, which utilizes Bootstrap for a modern and responsive layout.

## 2. Development Tools
- **IDE:** Visual Studio 2013 (version 12)
- **Programming Language:** C#
- **Database:** SQL Server 2014
- **Template:** Clean Blog (Bootstrap) | Source: [StartBootstrap - Clean Blog](https://startbootstrap.com/template-overviews/clean-blog/)

## 3. Features
- **Post Ownership:** Each post is tied to a registered user. Only the original author can edit or delete their own content.
- **User Authentication:** Includes login and password functionality, ensuring that certain actions can only be executed by registered users.
- **Responsive Design:** Built with a clean and user-friendly interface, designed for optimal performance across devices.
